KMP/Compose Multiplatform Member App · low-fidelity schermontwerpen + navigatieflow

Low-fidelity wireframes van de kernschermen van de native ledenportaal-app. Ze tonen de opzet en indeling per scherm (geen definitieve kleuren/typografie). De vier hoofdschermen — Home, Contributies, Mededelingen, Profiel — zijn bereikbaar via een onderbalk (bottom navigation); "Sign out" staat op het Profiel-scherm. De navigatieflow onderaan laat de samenhang zien. Bijbehorende user stories: zie user stories & DoD.
